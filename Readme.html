Brief Introduction

This is an analysis done as part of the Sustainable Forestry in Southern Sweden course in the Euroforester program at the SLU campus at the south of Sweden in Alnarp.

The data used for this analysis were provided mainly by the research centre at Tonnersjoheden and have been uploaded here.

Data description

  • block: experimental block

  • cutw: cultivar width

  • height: height of the plant

  • dia = diameter

  • clone: clone class

  • fert: Fertilized or not(1 = fertilized and 3 = control)

Height diameter relationship of the control and fertilized seedlings

     pch = c(16,17), col = c('red', 'blue'))
legend("topleft",
